## Kotlin Microservices

### The idea of this project is study kotlin and microservices with some strategies related to resilience
### load balance and so on...

### Requirements
- Java >= 21.0.2
- Docker Compose
- Datagrip or some sgdb

### How to start it

#### Firstly you can start a mysql container
- `docker-compose up`

### After that
You can import the project in your IDE like intelliJ and run each project individually
